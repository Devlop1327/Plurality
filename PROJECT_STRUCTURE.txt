💾 DATOS QUE SE GUARDAN
================================================================================

En LocalStorage se almacenan:

1. PROGRESO (plurality_progress)
   └─ educativo: 0-100%
   └─ juegos: 0-100%
   └─ simulador: 0-100%
   └─ evaluacion: 0-100%

2. PUNTOS DE JUEGO (plurality_game_score)
   └─ points: número total
   └─ level: calculado (points / 50 + 1)

3. RESULTADOS QUIZ (plurality_quiz_results)
   └─ Por módulo
   └─ Timestamp de cada intento

4. MODO OSCURO (plurality_dark_mode)
   └─ true/false

5. NOMBRE DE USUARIO (plurality_user_name)
   └─ Texto libre

================================================================================
📝 NOTAS IMPORTANTES
================================================================================

1. La app funciona completamente sin conexión a internet
   (excepto la descarga inicial de Tailwind CSS y iconos)

2. Los datos se guardan en el navegador local (no en servidor)

3. Si cambias de navegador, perderás el progreso guardado

4. Cada navegador tiene su propio almacenamiento aislado

5. Para una app real, se recomienda backends como:
   - Firebase
   - MongoDB
   - PostgreSQL
   etc.

6. El servidor.js es solo para desarrollo. Para producción,
   usa servidores como Nginx o Express.js
